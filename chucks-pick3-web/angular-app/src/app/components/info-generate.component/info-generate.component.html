How to Generate Picks:

By selecting 'View Picks' within mobile you will be allowed to generate new numbers for a specific time of day



You will then be taken to a seperate screen which will allow you to pick winning numbers from a specific time of day.



From here you can select the time of day for which you want to generate these new numbers for.
